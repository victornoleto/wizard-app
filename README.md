# Wizard App

## Configurar aplicação

Versões utilizadas:

```
node                         v24.7.0
npm                          11.5.1
angular                      20.1.0
rxjs                         7.8.2
typescript                   5.8.3
```

Instalar aplicação:

```
cd /var/www/wizard
https://github.com/victornoleto/wizard-app
cd wizard-app
npm install
```

Build da aplicação: `npm run build:prod`
